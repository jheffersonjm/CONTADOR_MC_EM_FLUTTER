# CONTADOR_MC

## Visão Geral
Projeto Flutter de contador (incrementar/decrementar), usado como base para praticar conceitos iniciais de Flutter/Dart.

## Status do Projeto
Em desenvolvimento

## Tecnologias
- Flutter
- Dart

## Requisitos
- Flutter instalado e configurado no ambiente
- Android Studio ou VS Code (recomendado)
- Emulador Android/iOS ou dispositivo físico

## Como executar
1. Clone o repositório:
   ```bash
   git clone https://github.com/jheffersonjm/CONTADOR_MC_EM_FLUTTER.git
   cd CONTADOR_MC_EM_FLUTTER
   ```

2. Acesse a pasta do app Flutter:
   ```bash
   cd contador
   ```

3. Instale as dependências:
   ```bash
   flutter pub get
   ```

4. Execute:
   ```bash
   flutter run
   ```

## Funcionalidades
- Incrementar contador
- Decrementar contador
- (Opcional) Zerar contador
